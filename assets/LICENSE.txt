# Assets for the MMS Player

The "Assets for the MMS Player" include two items

## The Gloria character

The file `avasag_avatar_casual.glb` contains the original Gloria virtual character in GLB format

## The default Blender scene

The file `defaults.blend` is the base Blender scene containing the imported Gloria character, default light setup, and some default render options.
